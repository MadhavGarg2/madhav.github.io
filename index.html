TheSharinganBot

What is "The Sharingan Bot"?

TheSharinganBot is a bot that is made specifically to focus on moderation and has commands like warn, warnings, stats, leaderboard, ping, kick, ban, mute, etc.

How To Use The Bot?

The prefix is s!. To use the mute command make a role in your server called "muted" remove the "" and keep all lowercase, give the muted role no permissions in any channel or if you wish you could control the mute command like you wish to.
